    Modules Robot Control MotorManager

    Thursday, October 2, 2014 10:36 PM

    motorManager

    Manages and sends commands to all arduinos.

    init

    Initializes everything.

    @Queue queue_in
    * Threadsafe queue for input commands
@Queue queue_out
    * Threadsafe queue for output
@String arm_port
    * Port for the arduino for the arm
@String drive_port
    * Port for the arduino contorlling the drive

    emergency_stop

    Shuts down all the motors

    read_inputs

    Starts the input reading thread

    run_input

    Runs a single input from a string. For debugging.

    @String input_string
    * Serialized command to run

    update_port

    Updates a port value. Should not be called externally.

    @String port
    * port to update
@String value
    * value to write to arduino for the port

    translate

    Turns a command into one the arduino understands. Returns it.

    @String port
    * port to update
@String value
    * value to write to arduino for the port

    shut_off

    Shuts off the robot. Cleans up values.
