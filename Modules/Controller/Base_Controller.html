    Base Controller

    The base controller class upon which custom controller schemes will inherit.

    Parameters

    init

    Initializes everything.

    @int id
    id of the controller. Gamepy gives you sequential: 0 is the first controller if you have 1 plugged in, 1 is the second if you have 2 controllers plugged in

    bind_button_up

    Will assign a function to the release of a button.

    @int button\_id
    button\_id is, according to the constants, the id of the button to assign a function
@function func()
    func is a zero argument function that gets called

    bind_button

    Will assign a function to the down of a button.

    @int button\_id
    button\_id is, according to the constants, the id of the button to assign a function
@function func()
    func is a zero argument function that gets called

    bind_button_hold

    Will assign a function to be called while a button is held down.

    @int button\_id
    button\_id is, according to the constants, the id of the button to assign a function
@function func()
    func is a zero argument function that gets called

    bind_axis

    Will assign a function that gets called when an axis changes value. Axes are the analog joysticks. There is a minimum detection, below which the magnitude is rounded to 0.

    @int axis\_id
    axis\_id is, according to the constants, the id of the axis to assign a function
@function func(int magnitude)
    func is a single argument function that gets called; magnitude is magnitude of joystick

    bind_hat(int hat_id, function func((int x, int y)))

    Will assign a function that gets called when a hat gets pressed. Hats are the cross-like buttons, with the up left right down arrows.

    @int hat\_id
    hat\_id is, according to the constants, the id of the hat to assign a function
@function func(int x, int y)
    func is a single argument function that gets called, and takes in a tuple (x, y), where x and y are the values of the hat
