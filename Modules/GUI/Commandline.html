    Modules GUI Commandline

    Thursday, October 2, 2014 10:08 PM

    Commandline

    Adds an input field for commands. Deprecated because the controller does that.

    Parameters

    init

    Initializes everything.

    @LabelFrame parent
    The root/parent. Usually self.root
@Queue queue_out
    Output queue for commands
@Queue queue_in
    Input queue for commands

    populate

    Populates the widgets with data. Change as you see fit.

    gui_loop

    Starts the gui thread.

    read_inputs

    Reads all input from the input queue.

    update_readout

    Updates readouts for motor values.

    @String motor
    Motor index to update
@String value
    New value to set motor to display

    update_controller

    Updates readouts for controller values.

    @String type
    If it is an axis, button, or hat
@String button
    Id of button to update
@String value
    New value to set controller to display

    is_active

    Checks if the gui is active

    destroy

    This kills the gui.
